Detailed Description

A variant type used to represent dynamically-typed values.

Value stores one of several kinds (null, bool, int, double, string, array, object, binary, datetime) and provides convenience accessors, mutators and JSON-like serialization.

The implementation uses std::variant for storage. Arrays and objects are reference-counted via shared_ptr to allow copy-on-write semantics.
