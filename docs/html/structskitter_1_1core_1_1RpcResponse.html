Detailed Description

Represents a JSON-RPC 2.0 response.

The 'id' field must be present and must be the same as the 'id' of the corresponding request. Exactly one of 'result' or 'error' must be present. If 'result' is present, it contains the successful result of the method call.

Creates a failure response with the given error and optional id.

If id is not provided, it will be set to null in the JSON output.

Parameters
errThe RpcError object containing the error code, message, and optional data to include in the response.
idvThe id value to include in the response. If not provided, it will be set to null in the JSON output. This allows creating responses for notifications if needed, although typically notifications should not have responses.
Returns
A RpcResponse object representing a failure response with the given error and id.
