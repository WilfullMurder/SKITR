Detailed Description

A simple TCP server that listens for incoming connections and processes JSON-RPC requests using a Dispatcher.

Runs in its own thread and can be started and stopped as needed. Binds to the specified port (or an ephemeral port if 0 is given) and handles each incoming connection by reading a line of input, parsing it as a JSON-RPC request, dispatching it to the Dispatcher, and sending back the JSON-RPC response. Designed to be simple and easy to use for testing and integration purposes.
